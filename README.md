# 📚 Appunti Programmazione 2

Appunti del corso di **Programmazione 2** scritte in Obsidian.  
Gli appunti coprono la programmazione orientata agli oggetti con Java.

---

## 🗂️ Struttura
```
Appunti-Programmazione-2/
├── Concetti/      # Definizioni e spiegazioni dei concetti chiave
└── Lectures/      # Note delle lezioni
```

## 📖 Argomenti

- Programmazione Orientata agli Oggetti (OOP)
- Classi, oggetti, costruttori, metodi
- Ereditarietà, polimorfismo, interfacce
- Classi astratte e modificatori di accesso
- Design Patterns (Singleton, Factory, Strategy)
- Principi SOLID
- Layout in memoria, Stack e Heap
