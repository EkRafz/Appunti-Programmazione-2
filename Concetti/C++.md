# C++

> [!info] Definizione
> **C++** è un linguaggio di programmazione multi-paradigma che supporta l'OOP ma, a differenza di [[Java]], permette l'[[Aritmetica dei puntatori]]. Questo significa che l'[[Incapsulamento]] può essere **bypassato** manipolando direttamente la memoria.

---

## Java vs C++: incapsulamento

| Aspetto | Java | C++ |
|---|---|---|
| Aritmetica dei puntatori | ❌ No | ✅ Sì |
| Incapsulamento bypassabile? | **No** | **Sì** |
| Campi `private` sicuri? | **Sempre** | **Bypassabili** con cast di puntatori |
| Gestione memoria | Garbage Collector | Manuale (o smart pointers) |
