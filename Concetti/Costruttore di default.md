

> Vedi [[Costruttore]].

Il **costruttore di default** è un [[Costruttore]] senza parametri generato **automaticamente** dal compilatore quando non si dichiara alcun costruttore. Inizializza tutti i [[Campo|campi]] ai loro valori di default.

```java
// Se la classe non ha costruttori:
public class TNT { }
// Il compilatore genera automaticamente:
// public TNT() { }
```

> [!warning] Se si dichiara **anche un solo** costruttore, il compilatore **non** genera più il costruttore di default!
