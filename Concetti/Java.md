

> [!info] Definizione
> **Java** è un linguaggio di [[Programmazione Orientata agli Oggetti|programmazione orientata agli oggetti]], **fortemente tipato**, modulare e portabile. È il linguaggio usato nel corso di Programmazione 2 per studiare i principi OOP.

---

## Caratteristiche principali

| Caratteristica | Dettaglio |
|---|---|
| **[[Programmazione Orientata agli Oggetti\|OOP]]** | Tutto è una [[Classe]] (non esistono metodi "scollegati") |
| **[[Incapsulamento\|Fortemente tipato]]** | Il compilatore verifica i tipi a compile time |
| **Modulare** | [[Package]], classi, interfacce |
| **Sicuro** | Niente [[Aritmetica dei puntatori]], [[Incapsulamento]] garantito |
| **Garbage collected** | La memoria viene gestita automaticamente |
