> [!info] Definizione
> In Java, **Object** e la classe base **di tutte le classi**. Se non specifichi `extends`, Java usa implicitamente `extends Object`.

---

## Metodi comuni

- `toString()`
- `equals()`
- `hashCode()`
- `getClass()`
